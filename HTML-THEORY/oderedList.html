This is Ordered List
  1. The HTML ol tag defines an ordered list
  2. An ordered list can be numerical or alphabetical.
  3. The type attribute of the ol tag, defines the type of the list item marker:
This is type A
  1. This is written in Uppercase alphabetical order A-Z
  2. similarly you can use type a for lowercase
This is type I
  1. The list items will be numbered with uppercase roman numbers
  2. similarly you can use i for lowercase
Type 1 is default
  1. The list items will be numbered with numbers (default)
